## Git

### Linux에 설치

Linux에서 패키지로 Git을 설치할 때는 보통 각 배포판에서 사용하는 패키지 관리도구를 사용하여 설치합니다. Fedora(또는 비슷하게 RPM 기반 패키지 시스템을 사용하는 RHEL, CentOS)에서는 아래와 같이 `dnf`를 사용합니다.

```bash
$ sudo dnf install git-all
```

Ubuntu등의 데비안 계열 배포판에서는 `apt`를 사용합니다.

```bash
$ sudo apt install git-all
```

다른 Unix 배포판에 설치하려면 http://git-scm.com/download/linux 에서 확인하시면 됩니다.

### Mac에 설치

Mac에 Git을 설치하는 방법 중에는 Xcode Command Line Tools를 설치하는 방법이 가장 쉽습니다. Mavericks(10.9)부터는 Terminal에 단지 처음으로 'git’을 실행하는 것으로 설치가 시작됩니다. 'git’이 설치돼 있지 않으면 설치하라고 안내해줍니다.

```bash
$ git --version
```

만약 Git이 시스템에 설치되어있지 않은 경우, 설치할 수 있도록 안내 메시지가 뜰 것입니다.

좀 더 최신 버전이 필요하면 바이너리 인스톨러로 설치할 수 있습니다. macOS용 Git 인스톨러는 Git 웹사이트에서 관리하고 있으며 http://git-scm.com/download/mac 에서 내려받습니다.

## Django

먼저 저장소로부터 프로젝트 소스코드를 받습니다.

### React와 공통되는 부분

```bash
shell> git clone 저장소.git
```

### 라이브러리 설치

**_manage.py_** 파일이 존재하는 위치에서 아래 명령을 실행하여 개발에 필요한 라이브러리를 설치합니다.

```bash
shell> pip install -r ./requirements/dev.txt
```

### 환경변수 설정

Root 경로에 **_.env_** 파일을 생성하여 필요한 환경변수를 셋팅합니다.

```yaml
DATABASE_URL=mysql://유저:패스워드@호스트/데이터베이스명

CORS_ORIGIN_WHITELIST=http://localhost:3000

ALLOWED_HOSTS=localhost,127.0.0.1
```

## React

먼저 저장소로부터 프로젝트 소스코드를 받습니다.

### Django와 공통되는 부분

```bash
shell> git clone 저장소.git
```

### 모듈 설치

**_package.json_** 파일이 존재하는 위치에서 아래 명령을 실행하여 개발에 필요한 모듈을 설치합니다.

```bash
shell> yarn
```

### 환경변수 설정

Root 경로에 **_.env_** 파일을 생성하여 필요한 환경변수를 셋팅합니다.

```yaml
REACT_APP_API_HOST=http://localhost:8000
```
